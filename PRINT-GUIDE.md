## 7. Technical Notes for the Printer

- **Color mode:** Spot color (single ink). No CMYK separation needed.
- **Mesh/screen:** Standard mesh for solid fills. No halftone dots.
- **Minimum detail:** The smallest icon features are approximately 0.5 mm at 100% scale. Use a mesh count that resolves this (230+ mesh for fine detail, 160 mesh for standard).
- **Registration:** Single color = single screen. No registration concerns.
- **Flash/cure:** Standard for the ink type used.
- **The rose icon is intentionally spaced further** from the other 6 icons (12 mm gap vs 5 mm). This is deliberate â€” do not "fix" the spacing.
- **The pistol icon is rotated 20Â°** and faces left. This is intentional.
